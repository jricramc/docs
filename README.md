# Nerd Documentation

Official documentation for [Nerd](https://nerdapp.ai) - the AI software builder that turns your ideas into complete platforms.

## What's inside

- **Introduction** - Learn what Nerd is and how it builds software
- **Getting Started** - Start building your first platform in minutes
- **How to Prompt** - Master the art of describing your platform effectively
- **Examples** - Real prompts that create stores, CRMs, ERPs, and more

## Key documentation principles

Our documentation focuses on:
- **Concise prompting** - Teaching users to write clear, direct prompts
- **No AI-generated prompts** - We explicitly discourage using ChatGPT to "improve" prompts
- **Real examples** - Actual prompts that create working platforms
